# 🐶 Pet Care Landing Page

Projeto desenvolvido a partir de um layout no Figma para praticar HTML e CSS.

## Sobre

Criei essa landing page como exercício para treinar a construção de layouts na prática.
A ideia foi pegar o design no Figma e recriar em código com o objetivo de praticar.

## Tecnologias usadas

- HTML
- CSS

## O que pratiquei

- Estrutura básica de uma página
- Organização com Flexbox
- Espaçamentos e alinhamento
- Estilização de botão

## Como abrir

Baixe o projeto e abra o arquivo index.html no navegador.
